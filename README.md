# Kaveri Family Restaurant POS System

A modern Point of Sale (POS) system built for restaurant management.

## Features

- Table management and order tracking
- Menu management with categories
- Order creation and billing
- User authentication and role-based access
- Reports and analytics
- Settings configuration

## Technologies

- **Frontend**: React, TypeScript, Vite
- **UI Components**: shadcn/ui, Tailwind CSS
- **State Management**: TanStack React Query
- **Routing**: React Router DOM
- **Charts**: Recharts

## Getting Started

### Prerequisites

- Node.js 18+ and npm

### Installation

```sh
# Clone the repository
git clone https://github.com/nampallyharish4/orderly-table.git

# Navigate to the project directory
cd orderly-table

# Install dependencies
npm install

# Start the development server
npm run dev
```

The app will be available at `http://localhost:5000`

## Build for Production

```sh
npm run build
```

The production build will be output to the `dist` folder.
